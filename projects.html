news & code

News & Code is a link-sharing site similar to Reddit or Hacker news focused on journalism and technology.

jump, buds.

This silly game is being developed in the open, using open source tools like Jekyll, jQuery, and Backbone.js. Feel free to fork the code and have fun.

seattle.io

seattle.io is a collection of data-oriented sites focused on Seattle, WA.

bandvideos.tumblr.com

bandvideos.tumblr.com: a tumblr blog for band videos. The custom tumblr theme is designed to show only video posts, and uses fitvids.js to fill the screen with the video.

Nearby in Seattle

Nearby in Seattle, a work-in-progress seattle.io project, is a mobile web app that shows nearby places of interest using data from SeattleWiki.

SeattleWiki

SeattleWiki is a community-authored wiki for Seattle, WA. Super Big Tree sponsors the project by providing a server, technical maintenance, and community organizing.

Moss Public Press

A news and technology startup based in the Pacific Northwest.

OlyLab

A sub-project of Moss Public Press, OlyLab provides a blog and wiki for Olympia, WA and the surrounding area.

Spindly Spud

Spindly Spud is a custom tumblr theme that allows infinite scroll, and utilizes Bootstrap for the base css styles.

The Dirt and The Water

An experiment in fictional world-building and web series production.
