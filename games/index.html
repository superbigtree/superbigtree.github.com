games.

Weird games made with javascript!

Save Christmas, Cool Chicken.

Seth Vincent and Brian Fullerton made this game during the second hogjam which took place at The Reactor in Seattle, WA. The game uses the 2d games toolkit crtrdg.js, which is one of our open-source projects.

Mossy Bellies

Originally created during the game-making competition Ludum Dare, this game is a work in progress, and utilizes the 2d games toolkit crtrdg.js, which is one of our open-source projects.

BlockSnot

A simple, jittery game made with coquette.js. Collect the squares, shoot the circles. Move with WASD keys, shoot with the mouse.

Read about making games with coquette.js on our blog.

Jump, Bud.

Currently a messy little prototype created with jQuery and Backbone.js, this game is slated for a complete re-do using crtrdg.js.
